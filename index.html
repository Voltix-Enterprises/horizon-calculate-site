Horizon Calculate

A lightweight Win32 calculator for Windows 2000 and newer

About

Horizon Calculate is a classic Win32 calculator written in C, designed to run on Windows 2000 through Windows 11 without any dependencies. It is fast, portable, and built with pure Win32 API calls.

Features

  • Pure Win32 API (no frameworks)
  • Runs on Windows 2000+
  • Small and fast executable
  • About dialog with system information
  • Classic Windows UI
